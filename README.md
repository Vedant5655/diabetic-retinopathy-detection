# Diabetic Retinopathy Detection

AI-powered web application for detecting diabetic retinopathy from retinal fundus images using Deep Learning.

## Features
- Upload retinal image
- Detect severity level
- Fast prediction
- User-friendly UI

## Tech Stack
- Python
- Flask
- TensorFlow / Keras
- OpenCV
- HTML/CSS/JavaScript

## How to Run
pip install -r requirements.txt
python app.py
